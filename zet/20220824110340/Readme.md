# Starting your day

I recommend starting the day at around 4am, following this format

* 4:00 - 4:20 - Prayer
* 4:20 - 4:40 - Meditation (20 mins recommended for a start)
* 4:50 - 5:30 - Book of sages
* 5:30 - 6:00 - Game of Go
* 6:00 - Reiterate your goals
* Practise Anki
* Practise Excercise
* Do some cleaning
